## 💻 Sobre o Projeto

Este projeto é uma aplicação web de **Lista de Tarefas** que permite ao usuário gerenciar seus afazeres de forma dinâmica. A aplicação salva as tarefas diretamente no navegador, garantindo que os dados não sejam perdidos mesmo após fechar ou atualizar a página.

Bom para aprender manipulação do DOM (Document Object Model) e guardar dados com Web Storage, acessando o Local Storage.

---

## ✨ Funcionalidades

- **Adicionar Tarefas:** Permite criar novas tarefas digitando no campo e clicando em "Register".
- **Validação de Entrada:** Impede o cadastro de tarefas com o campo em branco, exibindo um alerta ao usuário.
- **Listagem Dinâmica:** Renderiza a lista atualizada em tempo real na tela.
- **Excluir Tarefas:** Opção de remover tarefas individualmente da lista clicando em "Excluir".
- **Persistência de Dados:** Integração com o `localStorage` para manter a lista salva após recarregar a página.

---

## 🛠 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando apenas tecnologias web nativas:

* **HTML5:** Estruturação da página, campos de entrada e botões.
* **JavaScript (ES6):** Lógica da aplicação, manipulação do DOM e gerenciamento do estado da lista.
* **Web Storage (LocalStorage):** Persistência local dos dados em formato JSON e depois convertida para dados do JS.
